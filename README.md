# オタ活収納

## サイト概要
### サイトテーマ
オタクグッツの収納方法を検索したり、おすすめの収納を投稿したりできるサイト

### テーマを選んだ理由
自分がオタクなので収納をいつも検索していました。そんな中で既存の似たサイトはいろいろありますが、アニメ別やグッツ種類別に分かれているサイトはなくあれば便利だなと思い、このテーマにしました。

### ターゲットユーザ
グッツを集めている人

### 主な利用シーン
グッツの収納方法を検索したり、自分のおすすめを投稿したりできるサイト
